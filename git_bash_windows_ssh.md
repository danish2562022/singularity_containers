# Git Bash on Windows - Installation and SSH Tunnel Setup

A clean, step-by-step guide to install Git Bash on Windows and use it for SSH tunneling to Vera.

## Step 1: Download Git for Windows

1. Go to: https://git-scm.com/download/win
2. Click the download button
3. The installer will download automatically

## Step 2: Install Git Bash

1. Run the downloaded installer (`Git-2.x.x-64-bit.exe`)
2. Click "Next" through the installation wizard
3. **Important:** Keep default options selected (Git Bash will be installed)
4. Click "Install"
5. Wait for installation to complete
6. Click "Finish"

## Step 3: Open Git Bash

**Option A: From Start Menu**
- Click Windows Start button
- Type "Git Bash"
- Click on "Git Bash" application

**Option B: From Desktop/File Explorer**
- Right-click in any folder
- Select "Git Bash Here"

**Option C: From Desktop Shortcut**
- If created during installation, double-click the Git Bash icon on desktop

## Step 4: Verify Git Bash is Working

1. Git Bash window opens (black terminal window)
2. You should see a prompt like: `username@COMPUTERNAME MINGW64 ~`
3. Type `pwd` and press Enter
4. You should see your current directory path
